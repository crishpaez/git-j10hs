Working Directory

El area de trabajo corresponde a la carpeta donde inicializamos al repositorio de git, rara vez vamos a estar utilizando la linea de comandos y es basicamente donde vamos a crear, editar y eliminar los diferentes archivos de nuestro proyecto.

  • dir || ls :Muestra el contenido de la carpeta actual
  • mkdir <folder name> :Crea una carpeta con el nombre especificado.
  • cd <path> :Cambia la carpeta actual por la seleccionada.
  • git init :Inicializa un repositorio de git en la carpeta actual.
  • rm -rf .git :Elimina el repositorio de manera forzada
